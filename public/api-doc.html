/sendkeys 23 Sept 2020

To send secret key for covid positive patients.

  • Type: POST

  • Input: JSON Object 

    {
  secret_key: "red",
  timestamp: "123",
  auth_code:"123"
}

  • Output: String Message

    For success: successfully written

    For database issues: database error occured (In this case try once again.)

    For invalid auth code: authentication failed
